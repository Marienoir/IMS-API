# IMS-API
IMS is an Inventory Management System. The application uses Node.js and Postgresql for database management.

### Technologies Used
- Node.js
- Express.js
- Redis (Caching System)
- Node-Cron (Task Scheduling)

### Requirements
- Node
- npm

### Install Dependencies
```bash
npm install
```

### Environment variable
```bash
.env
```

### Run Application
```bash
npm run start
```
OR

```bash
npm run dev
```
